## Commit Prefix

### ルール
| type                  | emoji |
|-----------------------|:-----:|
| 機能追加、更新(update)       |  🔧   |
| バグ修正（fix)             |  🐛   |
| リファクタリング(refactoring) |  ♻️   |
| メモ(memo)              |  📝   |

### コミットメッセージの適用
```shell
$ cd <project>
$ cp .gitmessage.example .gitmessage
$ git config commit.template .gitmessage
```
